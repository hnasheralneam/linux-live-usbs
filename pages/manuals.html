User Manual

  1. 1) Make sure your computer is off
  2. 2) Insert USB flash drive into the computer
  3. 3) Press the power button to turn on the computer, and rapidly press the F9 key until the computer manufacturer logo shows up on the screen*
  4. 4) You should now be in the boot menu. Use the arrow keys to move to the list item that says "USB", "flash drive", or the name of the USB manufacturer (eg. PNY, SanDisk).
  5. 5) Some distrubutions (types of linux) come with a screen that will let you choose how to boot, if yours does choose "test/try [distro name]" and press enter.
  6. 6) Wait until it boots up,, then enjoy your new temporary linux environment!

Read more about Linux and live USBs


* HP uses the F9 key for the boot mentu, if you are using another device look [computer manufacturer name] bios/boot menu key

Note: some distro need secure boot to be disabled for booting off a USB

Manufacturer's Manual

Windows

  1. 1) Install Rufus media writer
  2. 2) Install the ISO for the desired distro, which you can find by searching "[distro name] live ISO/image".
  3. 3) Open Rufus. Insert the USB into the computer, and select it in the Rufus menu. Select the ISO you downloaded. Click "write".
  4. 4) Enjoy your new live USB!

Read more about Linux and live USBs

Linux

  1. 1) Install Fedora media writer (pre-installed on Fedora), Popsicle for writing to multiple USBs simulation,or an equivalent media writer tool.
  2. 2) Download the live boot ISO for the desired distro.
  3. 3) Open your media writer tool, and select the USB and ISO. Click write. This will take a while.
  4. 4) Enjoy your new live USB!

Read more about Linux and live USBs
